Fundamentos de CSS

CSS (Cascading Style Sheets) se usa para dar estilo y diseño a las páginas HTML.

Sintaxis básica de CSS

selector {
    propiedad: valor;
    otra-propiedad: valor;
}

Propiedades CSS comunes

Propiedad Descripción Ejemplo
color Color del texto color: #333;
background-color Color de fondo background-color: #f5f5f5;
font-size Tamaño de fuente font-size: 16px;
margin Margen exterior margin: 10px;
padding Relleno interior padding: 15px;
border Borde del elemento border: 1px solid #ddd;

Selectores CSS

Selectores de elemento

Aplican estilos a todos los elementos de un tipo específico.

p {
    color: blue;
}

Selectores de clase

Aplican estilos a elementos con un atributo class específico.

.destacado {
    background-color: yellow;
}

Selectores de ID

Aplican estilos a un elemento con un ID específico.

#header {
    font-size: 24px;
}

Diseño Responsive

El diseño responsive permite que tus páginas se adapten a diferentes tamaños de pantalla.

Viewport

Es esencial incluir esta meta etiqueta en el <head> de tu HTML:

<meta name="viewport" content="width=device-width, initial-scale=1.0">

Media Queries

Permiten aplicar estilos diferentes según las características del dispositivo.

@media (max-width: 768px) {
    /* Estilos para pantallas pequeñas */
    body {
        font-size: 14px;
    }
}

Unidades flexibles

Usa unidades relativas como %, vw, vh, rem y em para tamaños flexibles.

HTML Básico

HTML (HyperText Markup Language) es el lenguaje estándar para crear páginas web. Define la estructura y el contenido de una página.

Estructura básica de un documento HTML

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Título de la página</title>
</head>
<body>
    <!-- Contenido de la página va aquí -->
</body>
</html>

Elementos HTML comunes

Encabezados

Los encabezados van desde <h1> (el más importante) hasta <h6> (el menos importante). 

<h1>Título principal</h1>
<h2>Subtítulo</h2>

Párrafos y texto

El elemento <p> define un párrafo. Para formato de texto usamos <strong>, <em>, etc.

<p>Este es un párrafo. <strong>Texto importante</strong>. <em>Texto enfatizado</em>.</p>

Enlaces e imágenes

<a> crea hipervínculos. <img> inserta imágenes.

<a href="https://ejemplo.com">Enlace</a>
<img src="imagen.jpg" alt="Descripción">
